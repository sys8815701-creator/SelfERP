## 데이터 모델 계획

### 신규 모델
```
Vendor (확장)      — 이메일, 주소, 담당자, 은행정보 추가
AccountReceivable  — 미수금(외상매출금): 거래처, 금액, 만기일, 수금상태
AccountPayable     — 미지급금(외상매입금): 거래처, 금액, 만기일, 지급상태
TaxInvoice         — 세금계산서: 발행/수취 구분, 공급가액, 세액
BudgetItem         — 예산 항목: 연월, 계정과목, 예산금액
Estimate           — 견적서·청구서: 품목 라인, 합계, 발행상태
EstimateItem       — 견적서 품목 라인
```

### 기존 모델 활용
- `Journal` / `JournalLine` → 재무제표 자동 계산 기반
- `BankTransaction` → 현금흐름표 기반
- `CardSale` → 매출 데이터
