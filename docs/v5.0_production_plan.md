## 데이터 모델 계획

### 신규 모델
```
Item              — 품목: 원자재/반제품/완제품, 단위, 현재재고, 안전재고, 단가
BOM               — 자재명세서 헤더 (완제품 기준)
BOMLine           — BOM 구성 자재 (품목FK, 소요수량)
ProductionOrder   — 생산 지시서: 지시수량, 예정일, 상태
ProductionResult  — 생산 실적: 완료수량, 불량수량, 완료일
InventoryLog      — 입출고 이력: 유형(입고/출고/조정/생산소비), 수량, 단가
```

### 기존 모델 활용
- `Vendor` → 자재 공급처 연결
- `AccountPayable` → 자재 매입 대금
- `Employee` → 생산 인력
