# v9.0 최종 통합 모듈 계획서

**작성일**: 2026-07-02  
**범위**: v9.1 ~ v9.4  
**목표**: HR·회계·생산·유통 4개 모듈을 단일 뷰포인트에서 통합 관리

---

## 1. 개요

v1~v8에서 구축된 4개 도메인 모듈(HR, 회계, 생산, 유통)을 아우르는 전사 통합 레이어를 추가한다.  
사용자는 단일 화면에서 전체 사업 현황을 파악하고, 긴급 알림을 처리하며, 데이터를 내보낼 수 있다.

---

## 2. 버전별 구현 계획

| 버전 | 기능 | 방식 |
|------|------|------|
| v9.1 | 통합 대시보드 | 4모듈 KPI 카드 + 통합 알림 + 전체 메뉴 바로가기 |
| v9.2 | 전사 알림 센터 | 안전재고·연체 미수금·대기 수주 통합, 해제 가능 |
| v9.3 | 데이터 내보내기 | 12종 CSV 다운로드, 모듈 필터, 일괄 다운로드 |
| v9.4 | 권한 관리 | 3역할(관리자/매니저/뷰어) × 22메뉴 매트릭스 |

---

## 4. 데이터 흐름

```
통합 대시보드
  ├── GET /api/hr/employees/           → 재직 직원 수
  ├── GET /api/accounting/ar/summary   → 미수금 잔액·연체건수
  ├── GET /api/accounting/ap/summary   → 미지급금 잔액·연체건수
  ├── GET /api/production/summary      → 전체 품목 수
  ├── GET /api/production/safety-stock-alerts → 안전재고 부족 목록
  ├── GET /api/production/efficiency   → 평균 달성률
  └── GET /api/distribution/summary   → 대기 수주·배송 중 건수
```
