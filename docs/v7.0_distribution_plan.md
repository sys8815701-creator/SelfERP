## 데이터 모델 계획

### 신규 모델
```
Vehicle          — 배송 차량: 차량번호, 종류, 담당기사, 적재중량
SalesOrder       — 수주: 거래처FK, 납기일, 상태, 합계금액
SalesOrderItem   — 수주 품목: 수주FK, 품목FK, 수량, 단가
Delivery         — 배송 지시: 수주FK, 차량FK, 기사, 예정일
DeliveryItem     — 배송 품목 라인
Return           — 반품: 배송FK, 사유, 수량
```
